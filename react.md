# New [React](https://react.dev) project

## Create React project

```sh
$ PROJNAME=foo
$ npm create vite@latest $PROJNAME -- --template react
```

## ...as a new Git repo

```sh
$ git init
$ git add .
$ git commit -m "Initial commit"
```

## ...on GitHub

```sh
$ gh repo create $PROJNAME --private --source=.
$ git push
```
